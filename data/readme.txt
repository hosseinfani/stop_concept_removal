all_concepts: all unique concepts ids in the Twitter dataset
AllTweetDailyTimeseries: The occurrence number of tweets in each day within T days. 
AllTweetHourlyTimeseries: The occurrence number of tweets in each hour within 24*(T days). 
ConceptsDailyTimeseries: The occurrence number of concepts in all tweets of each day within T days. 
ConceptsHourlyTimeseries: The occurrence number of concepts in all tweets of each hour within 24*(T days). 

In our Twitter dataset: T=61 days (two months)
